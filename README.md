# JIConverter - Free Online PDF Tools

A free online PDF converter website with multiple tools for document conversion.

## Features
- PDF to Word Conversion
- Word to PDF Conversion
- PDF to Excel Conversion
- PDF to PowerPoint Conversion
- Image to PDF Conversion
- PDF to JPG Conversion

## How to Use
1. Visit the website
2. Select the tool you need
3. Upload your file
4. Convert and download

## Technologies Used
- HTML5
- CSS3
- JavaScript
- iLovePDF API for PDF to Word conversion

## Live Demo
Visit: https://yourusername.github.io/JIConverter/

## Setup for Development
1. Clone the repository
2. Open `index.html` in your browser
3. For PDF to Word, add your iLovePDF API key

## API Keys Required
For full functionality:
1. Get free API key from [iLovePDF Developer](https://developer.ilovepdf.com)
2. Add key in index.html (line where PUBLIC_KEY is defined)
